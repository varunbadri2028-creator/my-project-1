🧠 AI Human Behavior Predictor

📌 Overview

The AI Human Behavior Predictor is a simple rule-based system that predicts human behavior based on user inputs.
Instead of using machine learning, this project uses logical conditions (if-else rules) to simulate decision-making.

---

💡 Key Idea

Human behavior can sometimes be approximated using patterns and conditions.
This project uses predefined rules to analyze inputs and generate predictions.

---

⚙️ Technologies Used

- Python
- Basic Programming Logic (if-else conditions)

---

🔍 How It Works

1. The user provides input (like mood, situation, or choices).
2. The system processes the input using predefined rules.
3. Based on conditions, it predicts the behavior.

---

🚀 Features

- Simple and easy to understand
- Beginner-friendly AI concept
- Fast predictions (no training required)
- Demonstrates logical thinking approach

---

▶️ How to Run

1. Install Python
2. Download or clone this repository
3. Run the Python file

python filename.py


⚠️ Limitations

- Not a real machine learning model
- Works only on predefined rules
- Limited flexibility for complex behavior prediction
